Nested Divs

Intro

Nested divs zijn HTML-elementen die binnen andere div-elementen staan. Dit is een fundamenteel concept in HTML en CSS dat je helpt om complexe layouts te maken.

Wat zijn nested divs?

Een nested div is een <div> element dat zich bevindt binnen een ander <div> element. Dit creëert een hiërarchische structuur waarbij de binnenste div een "child" is van de buitenste div (de "parent").

HTML Structuur

De basis structuur van nested divs ziet er zo uit:

<div class="container">
    <div class="section">
        <div class="item">Item 1</div>
        <div class="item">Item 2</div>
    </div>
    <div class="section">
        <div class="item">Item 3</div>
    </div>
</div>

In dit voorbeeld:

  • container is de parent div
  • section divs zijn children van container
  • item divs zijn children van section (en grandchildren van container)

CSS Styling

Met CSS kun je nested divs op verschillende manieren stylen:

Display Properties

  • block: Elementen worden onder elkaar geplaatst
  • flex: Elementen worden naast elkaar geplaatst (met flexbox)
  • inline-block: Elementen worden naast elkaar geplaatst maar behouden hun blok-eigenschappen

Praktische Tips

Bij het werken met nested divs is het belangrijk om:

  • Logische structuur te behouden - elke div moet een duidelijke functie hebben
  • Semantische class namen te gebruiken (container, section, item)
  • Niet te diep te nesten - te veel niveaus maken code moeilijk te onderhouden
  • CSS selectors correct te gebruiken om specifieke nested elementen te targeten
