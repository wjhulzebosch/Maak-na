CSS Box Model

Het CSS box model is een van de belangrijkste concepten in CSS. Elk HTML element wordt behandeld als een rechthoekige box die bestaat uit vier lagen: content, padding, border en margin. De content is de daadwerkelijke inhoud van het element, padding is de ruimte tussen de content en de border, border is de lijn rondom het element, en margin is de ruimte buiten de border die het element scheidt van andere elementen.
