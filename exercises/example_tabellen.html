HTML Tabellen

Intro

HTML tabellen zijn een krachtige manier om gegevens in rijen en kolommen weer te geven. Ze zijn ideaal voor het presenteren van gestructureerde informatie zoals prijslijsten, roosters of vergelijkingen.

Basis tabel structuur

Een HTML tabel bestaat uit verschillende elementen:

  • <table>: Het hoofdelement dat de hele tabel bevat
  • <tr>: Table row - definieert een rij
  • <td>: Table data - bevat de daadwerkelijke celinhoud
  • <th>: Table header - bevat kopteksten (optioneel)

Table Headers (<th>)

Table headers worden gebruikt om kolommen of rijen te labelen. Ze zijn standaard vetgedrukt en gecentreerd, en helpen bij de toegankelijkheid van je tabel.

Verschil tussen <td> en <th>

  • <td>: Normale tabelcel met data
  • <th>: Header cel die kolommen of rijen beschrijft

HTML Code Voorbeelden

Hier zijn enkele voorbeelden van hoe je tabellen kunt opbouwen:

Basis tabel zonder headers

<table>
  <tr>
    <td>Red</td>
    <td>Blue</td>
  </tr>
  <tr>
    <td>Red</td>
    <td>Blue</td>
  </tr>
</table>

Tabel met headers

<table>
  <tr>
    <th>Red</th>
    <th>Blue</th>
  </tr>
  <tr>
    <td>Red</td>
    <td>Blue</td>
  </tr>
</table>

Praktische Tips

Bij het werken met HTML tabellen is het belangrijk om:

  • Consistente structuur te behouden - elke rij moet hetzelfde aantal cellen hebben
  • Headers te gebruiken voor betere toegankelijkheid en duidelijkheid
  • Logische volgorde aan te houden - headers bovenaan, data eronder
  • CSS te gebruiken voor styling in plaats van HTML attributen
