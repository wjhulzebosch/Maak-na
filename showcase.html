Maak-na-oefeningen Framework

Een interactief oefeningssysteem voor het leren van HTML, CSS en web development.

Dit framework biedt een gestructureerde manier om programmeeroefeningen te maken en te voltooien. Studenten kunnen oefeningen maken, direct feedback krijgen en hun voortgang bijhouden.

Hoofdfuncties

  • Meerdere oefeningstypes: HTML, CSS, en gecombineerde oefeningen
  • Interactieve preview: Zie direct het resultaat van je code
  • Automatische validatie: Krijg direct feedback op je antwoorden
  • Tab-gebaseerde organisatie: Oefeningen gegroepeerd per onderwerp
  • Voortgangsregistratie: Houd bij welke oefeningen je hebt voltooid
  • Flexibele input: Tekstvelden of afzonderlijke inputvelden

Documentatie

Wil je je eigen oefeningen maken?

Je kunt het project clonen vanaf de mondriaantin repository. Maak (met behulp van AI) een nieuwe exercise file aan en plaats het in de exercises folder. Je studenten kunnen dan op index.html?exercise=naam_van_oefening de oefening openen.

Hoe te gebruiken

  1. Klik op een van de oefeningen hierboven
  2. Lees de instructies in de tab
  3. Vul de ontbrekende code in
  4. Bekijk de preview om je resultaat te zien
  5. Vergelijk met het voorbeeld om te controleren of je antwoord correct is

Tip: Je kunt ook direct naar een specifieke oefening gaan door de URL parameter te gebruiken: index.html?exercise=naam_van_oefening
